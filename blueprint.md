
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, and it uses modern web standards like Web Components to create a clean and maintainable codebase.

## Features

*   Generate 6 unique random numbers between 1 and 45.
*   Display the generated numbers in a visually appealing way.
*   A "Generate" button to trigger the number generation.
*   Responsive design that works on both desktop and mobile devices.
